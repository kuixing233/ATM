## 二 程序的架构设计
### 1、程序设计的好处
    1）思路清晰
    2）不会出现写一半代码推翻重写
    3）方便自己或以后的同事更好的维护
    
### 2、三层架构设计的好处
    1）把每个功能都分层三部分，逻辑清晰
    2）如果更换不同的用户界面或数据储存机制
    不会影响接口层的核心逻辑代码，扩展性强
    3）可以在接口层准确的记录日志与流水
